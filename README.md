## Installation

```bash
$ npm install
```

## Setup database

Make shure Docker is running and:
```bash
$ npm run db:dev:restart
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```

## Test

```bash
# e2e tests with live reload
$ npm run test:e2e
```
